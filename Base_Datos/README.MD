# Base de Datos

En esta seccion encontrará todo lo relacionado con las versiones disponibles de la base de datos de Dinamo.

La base de datos de Dinamo es de tipo relacional, por lo que dentro de los diferentes scripts encontrará lo siguiente:

- Script de instalación
- Base de datos
- Tablas
- Procedimientos almacenados
- Disparadores
- Vistas

Actualmente la base de datos de Dinamo esta escrita puramente para `SQL SERVER`, hay un versión para `MYSQL`, pero no es estable y esta desactualizada, lo recomendable es usar el la CLI de prisma para hacer una migración de SQLSERVER a MYSQL y otr SGBD en lo que se actualiza la version de MYSQL.

**NOTA: Tenga en cuenta que si no esta a gusto con las versiones disponibles de la base de datos de Dinamo, puede hacer una migración a su SGBD de preferencia, pero tenga en cuenta que hasta que no se finalice la base de Dinamo, no habrán actualizaciones para otrs SGBD**

**NOTA: Tenga en cuenta que hay ciertos objetos que no podrá ejecutar si usa un SGBD diferente a SQL SERVER, como por ejemplo los procedimiento de envío de correos, por tal motivo es necesario que tenga en cuenta estas acciones para que el aplicativo no falle**
